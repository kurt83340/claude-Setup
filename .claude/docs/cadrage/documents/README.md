# documents/ — Pièces reçues du client

Archive ici tout ce que le client te file : PDF, Excel, captures, exports, doc tierce.

## Naming

`YYYY-MM-DD-description-courte.ext`

Exemples :

- `2026-05-18-cdc-client.pdf`
- `2026-05-20-export-exemple.xlsx`
- `2026-05-21-doc-sap-service-layer.pdf`

## Règles

- ❌ **PAS de données sensibles non anonymisées** (RGPD)
- ❌ **PAS de credentials** en clair (à stocker hors repo)
- ✅ Anonymiser les exports clients avant commit (remplacer noms par `Client A`, `Client B`)
- ✅ Si fichier trop sensible → stocker hors repo, juste un placeholder ici avec note
- ✅ Garder l'historique (`v1.pdf`, `v2.pdf`) plutôt qu'écraser
